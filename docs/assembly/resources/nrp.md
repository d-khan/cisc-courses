# National Research Platform (NRP)

## What is NRP?

The National Research Platform (NRP) is a community-owned cyberinfrastructure that provides researchers, educators, and
students with access to cloud computing resources.

---

## Why Use NRP?

NRP provides:

- Free Jupyter Notebooks
- Cloud Computing Resources
- GPU Access
- Large Data Storage
- AI and Machine Learning Tools

---

## Access NRP

Visit:
[NRP](https://nationalresearchplatform.org/)

---

## NRP JupyterHub

Launch interactive notebooks directly from your browser.

### Supported Languages

- Python
- R
- Julia

Examples include:
